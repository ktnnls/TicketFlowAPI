# TicketFlowAPI

API REST desenvolvida com Spring Boot para gerenciamento de chamados e suporte ao cliente.

## 🚀 Tecnologias

* Java 21
* Spring Boot
* Spring Security
* JWT Authentication
* PostgreSQL
* Swagger/OpenAPI
* Maven

## 📋 Funcionalidades

* Autenticação com JWT
* Cadastro e gerenciamento de usuários
* CRUD de tickets/chamados
* Controle de status e prioridade
* Arquitetura REST organizada
* Integração com banco de dados PostgreSQL

## 📁 Estrutura do Projeto

```bash
src/
 ├── controller
 ├── service
 ├── repository
 ├── entity
 ├── dto
 ├── config
 └── security
```

## ▶️ Como executar

### Pré-requisitos

* Java 21
* Maven
* PostgreSQL

### Executando o projeto

```bash
mvn spring-boot:run
```

## 📌 Objetivo

Projeto desenvolvido para estudos de arquitetura backend, autenticação JWT e construção de APIs corporativas utilizando Java e Spring Boot.
